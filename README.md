Бот-планер для организации дел
Проект студенток ОП ФиКЛ НИУ ВШЭ Федоровой Софьи, Уваровой Александры, Дорофеевой Дарины и Радиной Ульяны
Описание проекта (@planner_ficl_bot): Telegram-бот, который автоматизирует планирование задач и помогает распределять время за счёт алгоритмов и напоминаний. Бот позволяет пошагово добавлять задачи в список дел. 
Пользователь вводит название задачи, выбирает категорию задачи из предложенного списка, выбирает дедлайн выполнения и длительность самой задачи, а также, по желанию, добавляет список вещей, которые нужно взять с собой или адрес мероприятия.
Список задач можно увидеть с помощью кнопки «Мои задачи», в нём уже автоматически проставлены приоритеты. Задачи можно удалить или пометить, как выполненные. Позже бот отправляет уведомление о дедлайне и оповещает пользователя, если задача просрочена. 
Структура репозитория: файл project.py и ReadMe (https://github.com/mockingirl9/project)
Для того, чтобы запустить код нужно ввести pip install py TelegramBotAPI schedule в терминал питона. Токен бота: @BotFather
Нейросеть Дипсик использовалась для того, чтобы определить, какие библиотеки необходимо было импортировать (промт: «Напиши список импортов для Python-бота в Telegram с базой данных, клавиатурами и фоновым планировщиком задач».
