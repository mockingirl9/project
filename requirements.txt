Импортируемые библиотеки:
import telebot – Основная библиотека для Telegram Bot API
from telebot import types – Типы для клавиатур и кнопок
import sqlite3 – Встроенная БД для хранения задач
from datetime import datetime, timedelta – Работа с датами и временем
import threading – Для запуска напоминаний в фоновом потоке
import time – Паузы в планировщике
import schedule – Планировщик периодических задач

База данных:
Таблица tasks содержит все поля задачи:
id — уникальный идентификатор
user_id — Telegram ID пользователя
title, category, deadline, duration (в минутах)
priority (1-3, где 3 — высший)
status (active/completed/overdue)
address и items_to_bring — дополнительная информация
